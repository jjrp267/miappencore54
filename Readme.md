# Symfony and React

app built with react and Symfony 5.4

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install foobar.

```bash
composer install
yarn install
```

## server

```server
symfony server:start

yarn run dev-server
```
